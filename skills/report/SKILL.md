## When to invoke

The agent should reach for this skill when:
- A cycle completes (pass or fail)
- Evidence needs human-readable summary
- Burn report needs generation

## Report Sections

1. **Header** — Task, Task ID, Cycle #, Status, Timestamp
2. **SLA** — Wall clock, tool calls, violations
3. **Evidence** — Per-gate: command, expected, actual, status, duration
4. **Footer** — Generated by fuzzybee v0.1.0

## Burn Report

Daily/weekly aggregation:
- Total cycles, pass rate, avg duration
- Total tool calls, most common problem type
- Pattern clusters from learnings
